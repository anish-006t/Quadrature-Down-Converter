# Quadrature-Down-Converter
Designed and implemented a quadrature down converter in LTSpice for RF signal processing. The system uses a MOSFET-based mixer, Bubba oscillator (100 kHz sine/cosine with ~90° phase shift), and RC low-pass filters (~2 kHz cutoff) to perform frequency translation (f_in − f_osc). Verified performance using transient and FFT analysis.
